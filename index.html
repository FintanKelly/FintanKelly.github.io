    Section 1: MQ2Melee

    MQ2Melee automation focuses mainly on two elements: holyshits and downshits.

    Holyshits: These run when you are in combat.

    Downshits: These run when you are out of combat.



    However before we get to these, I want to make sure that you have a .ini file ready to edit. These files have a naming convention: '<Server Short Name>_<Character Name>.ini'

    You can force MQ2Melee to make a .ini file by typing in '/melee save'
